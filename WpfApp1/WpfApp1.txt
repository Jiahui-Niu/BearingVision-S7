## WpfApp1 重新开发说明

开发日期：2026-06-21
开发依据：4张截图 + ICPlatformTools已恢复代码 + git历史记录

### 编译前置条件

1. 安装 VisionMaster 4.4.3（海康机器人视觉算法平台）
   - 安装后会自动注册 VM.Core, VM.PlatformSDKCS 等程序集
   - log4net.dll 路径：C:\Program Files\VisionMaster4.4.3\Development\V4.x\ComControls\Assembly\

2. HslCommunication.dll
   - 放置于 WpfApp1\bin\x64\Debug\ 目录
   - 版本支持 S7 1200/1500 和 Omron FINS TCP

3. Newtonsoft.Json.dll
   - 已包含在 ICPlatformTools\Dll\ 目录

### VM SDK 集成说明（重要）

MainViewModel.cs 中的 TriggerVMCamera() 方法包含 TODO 注释。
需要根据实际VM工程配置修改以下内容：
- Group名称（当前占位为 "Cam1" 至 "Cam6"）
- 全局变量名称（触发信号、检测结果）
- 具体的 VM SDK API 调用

参考 VM SDK 文档确认以下API：
- 加载方案：MVisionMaster.Instance.Solution.Load(path)
- 执行流程：solution.GetProcedure("GroupName").Execute()
- 读取结果：global variable module API
